# hexo-theme-slience

A slience theme for hexo.

## Demo

[Woonden Bar](http://varzy.me/blog)

## Installation

``` shell
$ git clone https://github.com/hexojs/hexo-theme-slience.git themes/landscape

$ npm i -D hexo-renderer-sass
```

## Enable

Modify `theme` setting in `_config.yml`.

## Update

```
$ git pull
```
