# hexo-theme-slience

A silent theme for hexo.

## Demo

[Woonden Bar](http://varzy.me/blog)

## Installation

``` shell
$ git clone git@github.com:varzy/hexo-theme-slience.git themes/hexo-theme-slience

$ npm i -D hexo-renderer-sass
```

## Enable

Modify `theme` setting in `_config.yml`.

## Update

```
$ git pull
```
